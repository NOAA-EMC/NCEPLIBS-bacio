# BACIO
-------

Performs I/O for the NCEP models.

## Getting Started


### Prerequisites

Compilers: GNU | Intel | Clang | AppleClang | PGI


### Installing

```
mkdir build
cd build
cmake -DCMAKE_INSTALL_PREFIX=/path/to/install /path/to/NCEPLIBS-bacio
make -j2
make install
```
